Утилита CHR96


    В МК 85 символ под номером 96 является переназначаемым и задать его можно при помощи команды LETC с закодированной строкой. Для описания каждой линии (5 пикселей) в строке используется один символ от 0 до V. В инструкции по эксплуатации пользователям предлагалось производить кодирование пикселей в строку вручную, при помощи таблицы:


    Символ 96 очень полезен для вывода графики в программах и играх, но ручное кодирование весьма неудобно. Утилита позволяет в графической форме задать символ и получить его код. Достаточно лишь кликнуть по нужным пикселям на матрице, после чего в строке ввода будет готовый код. В эту строку можно ввести и ранее закодированный символ, в таком случае на матрице появится его графическое представление. Кнопки "Copy" и "Clear" служат для копирования в буфер обмена и очистки строки (а вместе с ней и матрицы) соответственно.
    Кнопка "Send to MK" позволяет передать находящийся в строке код в виртуальный микрокомпьютер с помощью эмуляции нажатий клавиш. Поскольку код является строкой, в микрокомпьютере он должен быть обрамлен кавычками ("), поэтому для удобства пользователя с помощью опции "in quotes" при посылке строки она автоматически будет помещена в кавычки.

    ВНИМАНИЕ! Для корректной работы посылки строки необходимо, чтобы микрокомпьютер находился в нормальном режиме работы (в режиме ошибки ввод невозможен), не был активен расширенный режим (EXT) и модификаторы S и F. В противном случае передаваемый код будет искажен.

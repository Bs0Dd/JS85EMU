Virtual microcomputer


    The appearance of the microcomputer is similar to the real one, with the exception of the "initial settings" button, which is located on the back side of the device, therefore, for convenience, it is placed on the front side in the emulator.
    The black symbols to the left of the red inscriptions in the original are on a separate overlay supplied with the device. The display of these symbols can be turned on or off using the corresponding item in the options panel.

Options panel


    The options panel consists of 6 sections and is intended for configuring the emulator. Here you can view the amount of available memory for BASIC programs. Pause or restart the emulator, show or hide overlay symbols, enable or disable vibration when pressing keys (mobile devices only). It is possible to change the size of RAM, load RAM or ROM from files, save the contents of RAM, a list with variables, a list of BASIC programs into files, which can also be loaded into the virtual microcomputer itself. Here you can adjust the operating speed for standard and turbo modes and switch between them. Some debugging and experimental functions are also available, and you can also open a window with a layout diagram, a help window, the "About" page, and show/hide the Debugger or the CHR96 utility.

CHR96 utility


    In MK 85, symbol number 96 is reassignable and can be set using the LETC command with an encoded string. One character from 0 to V is used to describe each line (5 pixels). The utility allows you to specify a symbol in graphical form and obtain its code, which can then (via pressing emulation) be transferred to a virtual microcomputer. It is also possible to decode the string, obtaining a graphical form of the symbol.

Debugger


    A debugger is a debugging complex, rather useful for those who want to modify the MK 85 firmware or even create their own. Displays the state of registers (including special ones) and allows you to change their values. You can control the operation of the device (the "pause" and "restart" keys are duplicated from the options panel), debug the firmware in step-by-step mode, and set a breakpoint. There is a RAM editor, with the ability to view content in "bytes" and "words" (order from low to high) modes. The complex also includes a lite-disassembler for viewing currently performed operations. If desired, any command can be replaced using lite-assembler. You can switch display and input modes between hexadecimal and octal.

CHR96 utility


    In MK 85, symbol number 96 is reassignable and can be set using the LETC command with an encoded string. One character from 0 to V is used to describe each line (5 pixels). The manual suggested that users encode pixels into a string manually using a table:


    The 96 character is very useful for displaying graphics in programs and games, but manual coding is very inconvenient. The utility allows you to specify a symbol in graphical form and get its code. You just need to click on the desired pixels on the matrix, after which the ready-made code will be in the input line. You can also enter a previously encoded symbol into this line, in which case its graphical representation will appear on the matrix. The "Copy" and "Clear" buttons are used to copy to the clipboard and clear the line (and with it the matrix), respectively.
    The "Send to MK" button allows you to transfer the code in the input line to a virtual microcomputer using keystroke emulation. Since the code is a string, in the microcomputer it must be surrounded by quotation marks ("), therefore, for the convenience of the user, using the "in quotes" option, when sending a string, it will automatically be placed in quotation marks.

    ATTENTION! For the correct operation of sending a string, it is necessary that the microcomputer be in normal operation mode (in error mode input is not possible), the extended mode (EXT) and modifiers S and F were not active. Otherwise, the transmitted code will be distorted.

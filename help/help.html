    JS85EMU is an emulator of the Soviet microcomputer/microcalculator/personal computer (these three types are mentioned on the box, in the instruction or on the device itself) Elektronika MK85(M).

    The task of the emulator is to emulate the device as accurately as possible, taking into account its technical features. This emulator introduces software switch and Turbo mode emulation for the first time. It also contains a PDP-11 debugger (microcomputer processor architecture), a utility for encoding pixels into a string for character 96 (LETC command). A system for decoding BASIC programs from internal format to text and vice versa also has been implemented. You can view the values of variables in memory.

    The help describes the features of working with the emulator and its subsystems. If you have not previously had experience working with the MK 85 (real or through another emulator), it is first recommended that you read the original instruction manual, where you can find basic information about the microcomputer and how to work with it. Unfortunately, there is only a Russian version of the manual, which makes it difficult for foreign users to understand. The translation of images does not provide 100% accuracy, so for simplicity, you can also study the manual of the "prototype" device - Casio FX-700P. Although the MK85 has additional commands and some differences in the operation of existing ones, most of the MK85 documentation is clearly borrowed from the FX-700P documentation.
